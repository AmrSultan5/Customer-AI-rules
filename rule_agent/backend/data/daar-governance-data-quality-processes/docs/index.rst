Welcome to Data Mesh Data Quality Application documentation!
====================================================

Data Mesh Data Quality Application keeps all processes responsible for read data from Curated layer from Data Lake,
transformation by YAML files and write datato Datalake. 
Currently process repository doesn't have any custom classes and whole logic
is defined in meta-configuration files under governance_data_quality_processes/configs
